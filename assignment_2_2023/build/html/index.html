” .. module::Node_A User interface

platform:

Unix

synppsis:

This module is the user interface for the robot controller. It allows the user to set a goal for the robot to reach and to cancel the goal if needed. The user can also request the robot’s current position and velocity.

Module author: Yazan Kayali

subscribes to:

/odom

Publishes to:

/Robot_state

client :

/reaching_goal

scripts.node_a.goal_cancel_(client)[source]

Cancelling goal function

This function cancels the goal that has been sent to the action server

Parameters:

client (SimpleActionClient) – the client object used to send the goal to the action server

scripts.node_a.main()[source]

Main function

This function initializes the node, creates the SimpleActionClient, and subscribes to the odometry topic. It also contains the main loop that allows the user to set a goal for the robot to reach and to cancel the goal if needed. The user can also request the robot’s current position and velocity.

scripts.node_a.odom_callback(odom_msg, robot_state_pub)[source]

Odometry callback function

This function extracts the position and velocity information from the odometry message and publishes it on the RobotState topic

Parameters:

  • odom_msg (Odometry) – the odometry message containing the position and velocity information

  • robot_state_pub (Publisher) – the publisher object used to publish the position and velocity information on the RobotState topic

” .. module::Node_B last_target_service_node

platform:

Unix

synppsis:

This module is a ROS node that subscribes to the ‘/reaching_goal/goal’ topic and stores the last target coordinates set by the user. It also provides a service ‘/last_target’ that returns the last target coordinates when called.

Module author: Yazan Kayali

subscribes to:

/reaching_goal/goal

publishes to:

None

services:

/last_target

clients:

None

scripts.node_b.callback(msg)[source]

Callback function

This function is called whenever a message is published to the ‘/reaching_goal/goal’ topic. It stores the last target coordinates set by the user.

Parameters:

msg (PlanningActionGoal) – The message published to the ‘/reaching_goal/goal’ topic. It contains the target coordinates set by the user.

scripts.node_b.last_target_handler(req)[source]

Service handler function

This function is called whenever the service ‘/last_target’ is called. It returns the last target coordinates set by the user.

Parameters:

req (SetBoolRequest) – The request message sent to the service ‘/last_target’. It contains the active state of the service.

scripts.node_b.main()[source]

Main function

This function initializes the node ‘last_target_service_node’, subscribes to the ‘/reaching_goal/goal’ topic, and creates the service ‘/last_target’.
